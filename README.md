# 🐷 AI 存钱小帮手

一个**手机 App 风格**的网页版智能存钱助手。默认货币为 **RM（马来西亚林吉特）**，可在设置中更改。

它有两种用法：
- **纯本地（零配置）**：直接打开就能用，数据只存在你自己的浏览器里，离线可用，AI 用内置智能逻辑。
- **全栈（部署后）**：手机号 + 密码登录、数据云端同步（**换手机/重装不丢**）、「小帮手」接入**真·Claude AI** 对话。部署只需照着 [`DEPLOY.md`](DEPLOY.md) 走一遍（约 15 分钟，免费）。

## ✨ 功能

| 模块 | 说明 |
| --- | --- |
| 🎯 **存钱目标管理** | 设定目标（金额 + 期限 + 图标 + 每月计划），进度条、剩余金额、按节奏/计划预测达成日期一目了然；支持多目标切换与**归档**。 |
| 📒 **记账 / 存款记录** | 一键记录每笔存入 / 取出，支持**点按编辑**、来源快捷标签（工资/红包/省下的…）、按天分组、按时间和类型筛选。 |
| 📊 **统计与洞察** | 「最近 30 天累计」曲线 + 「近 6 个月存入」柱状图，以及本月 vs 上月对比、日均、当前/最长连续天数、已达成目标数等洞察。 |
| 🐷 **AI 省钱建议与鼓励** | 内置智能逻辑生成个性化建议、提醒与鼓励；登录后「小帮手」自动升级为**真·Claude AI** 对话（结合你的真实数据回答），离线/未配置时优雅回退到内置逻辑。 |
| 🏆 **存钱挑战玩法** | 52 周存钱法、12 个月递增、每日打卡等趣味挑战，完成的存入会**自动记账到目标里**。 |
| 👤 **账号 + 云同步** | 手机号 + 密码注册/登录（密码加密存储），数据加密同步到云端，换手机或重装登录即可恢复。 |
| 🌙 **深色模式** | 支持「跟随系统 / 浅色 / 深色」，全应用配色自适应。 |
| 🔔 **每日提醒** | 当天还没存钱时首页横幅提醒，并可选开启浏览器通知。 |
| 💾 **数据安全** | 未登录时仅存本机；支持导出/导入备份，导入自动做白名单校验，杜绝脏数据/注入。 |

## 🚀 使用方法

**方式一：直接打开（纯本地）**
双击 `index.html` 即可在浏览器中使用，所有本地功能完整可用。

**方式二：本地起服务（含后端，可测登录/同步/AI）**

```bash
cd "claude ai_project"
python3 server.py          # 打开 http://localhost:8000
```

仅用标准库即可启动（数据存本地 `data/app.db`）。想本地就用真 AI：
`pip3 install -r requirements.txt` 后用 `ANTHROPIC_API_KEY=sk-ant-... python3 server.py` 启动。

**方式三：部署上线（账号 + 云同步 + 真 AI + 手机安装）**
照着 [`DEPLOY.md`](DEPLOY.md) 一步步来即可（Render 免费部署，约 15 分钟）。

> 手机上用：在 Safari / Chrome 打开网址后，点「分享 / 菜单 → 添加到主屏幕 / 安装应用」，即可像原生 App 一样全屏使用，并支持离线。

## 📁 文件结构

```

icon.svg         # 应用图标
DEPLOY.md        # 部署 + 手机安装教程
```

## 🔒 隐私与安全

- **未登录**：数据通过浏览器 `localStorage` 仅存本机，不上传。
- **已登录**：数据同步到你自己部署的后端数据库；API Key 只在服务端，绝不进前端；密码用 PBKDF2 加密存储，登录态用 JWT。
- 可在「设置 ⚙️」中随时导出 / 导入 / 清空数据。
